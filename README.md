# ⭐ Age Calculator App

Aplicación web responsive inspirada en el reto de Frontend Mentor **"Age Calculator App"**.

El objetivo fue desarrollar una calculadora de edad capaz de validar fechas ingresadas por el usuario y calcular con precisión los años, meses y días transcurridos desde la fecha de nacimiento, implementando buenas prácticas de JavaScript, validación de formularios y una interfaz responsive utilizando HTML, Tailwind CSS y JavaScript.

---

## 📌 Descripción

Este proyecto consiste en una aplicación que calcula la edad exacta de una persona a partir de su fecha de nacimiento.

Se desarrolló con un enfoque mobile-first y cuenta con un formulario que valida la información introducida por el usuario antes de realizar el cálculo.

La aplicación verifica que los campos no estén vacíos, que los valores ingresados sean válidos, que la fecha realmente exista y que no corresponda a una fecha futura.

Una vez validada la información, calcula correctamente la diferencia entre la fecha de nacimiento y la fecha actual, mostrando el resultado en años, meses y días.

La interfaz fue construida utilizando Tailwind CSS y JavaScript, priorizando una estructura limpia, código reutilizable y una experiencia de usuario intuitiva.
